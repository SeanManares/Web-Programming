HTML and CSS Quick-Reference Guide

This website covers all of the tags that have been discussed over the course of web programming with description of how to use them and code examples.

HTML Tags

<h1> to <h6>

It defines html headings. <h1> stands out the most as it is usually the largest font on the page, <h6> on the other hand, is the smallest heading tag.

<h5>Heading</h5>
This is how <h5> heading looks like

<p>

It defines a paragraph.

<p>This is a paragraph.</p>

<marquee>

It is a tag that defines a scrolling text area in the html document that moves across the page in a horizontal or vertical direction.

<marquee>Hello World</marquee>

Output:

Hello World

<table>

This creates a table in the webpage.

<table border="1px">     
    <tr>                                   
        <th>Header 1</th>
        <th>Header 2</th>            
    </tr>
    <tr>
        <td>Data 1</td>              
        <td>Data 2</td>
    </tr>
</table>

Output:

Header 1 Header 2
Data 1 Data 2

<img>

This defines an image in the webpage.

<img src="image.jpg" width="100px" height="100">

Output:

<ul> and <li>

This defines unordered list and list items.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol> and <li>

This defines ordered list and list items.

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

<div>

Defines a division or section in an HTML document.

<div>This is a division.</div>

<span>

Defines a section in a document, mostly used for styling.

<span style="color: red;">This is a span.</span>

Output:

this is a span

<form>

It is used to create an HTML form for user input.

<form action="/submit" method="post">
    <input type="text" name="name">
    <input type="submit" value="Submit">
</form>

Output:

CSS Properties

color

It sets the color of a text.

id {
    color: red;
}

Output:

Color red

padding

It is used to create space around an element's content, inside of any defined borders.

id {
    padding: 10px;
}

Output:

Padding

float

The CSS float property is used for positioning and formatting content. For example, the picture can be positioned at the left of the text. 

id {
    float: left;
}

Output:

This is an image.

align

Align specifies how flex lines are distributed along the cross axis in a flexbox container. 

id {
    text-align: center;
}

Output:

The text is centered.

overflow

overflow controls what happens to content that is too big to fit in the given area.

id {
        overflow: auto; 
        width: 160px;
        height: 150px;
        background-color: aqua;
    }

Output:

This is an example of overflow with a scroll bar, it also have the following values: visible, hidden, and auto. Visible is the default, the overflow is not clipped. Hidden is the overflow is clipped, and the rest of the content will be invisible. Scroll adds a scrollbar even if not needed, while auto adds scroll bar only when necessary.

background-color

It specifies the background color of an element.

id {
        background color
    }

Output:

This is background color of an element

border-style

it specifies what kind of border to display. There are different border style but one example is the Dotted border

id {
        border-style: dotted;
    }

Output:

This is the dotted border style

background-image

It specifies an image to use as the background of an element, usually put in the head tag.

id {
        background-image: url("image.jpg");
    }

Output:

This area shows a background image
